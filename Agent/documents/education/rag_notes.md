# RAG NOTES

- Always distinguish between workman and non-workman before answering.
- Check whether Labour Codes or legacy Acts apply.
- Verify State Shops & Establishments Act where applicable.
- Mention applicable Labour Code alongside older legislation.
- Confirm latest State notification before relying on numerical limits.
- Verify Labour Code implementation status in the concerned State.
- Mention BSA when electronic evidence is involved.
- Mention BNS where cyber offences overlap with employment disputes.
