# Retrieval Notes

Always distinguish:

- Civil Remedy (Section 43)
- Criminal Remedy (Section 66 onwards)

Always recommend:

1. Call 1930
2. Report at cybercrime.gov.in
3. Inform bank
4. Preserve evidence
5. File FIR

Do not recommend Section 66E unless privacy or intimate images are involved.

Use:

66C → Identity Theft

66D → Cheating by Personation

43 → Civil Compensation

43A → Data Security

67 → Obscene Content

67A → Sexually Explicit Content

67B → Child Sexual Abuse Material

Return only laws relevant to the user's facts.
